# File-System-Automator
Overview This Python utility was developed to automate the organization of messy directories (like Downloads or Desktops). It scans a target folder, identifies file types by their extensions, and moves them into categorized sub-folders to maintain a clean local infrastructure.

What This Project Demonstrates:

• Automation: Using Python to handle repetitive system-level tasks.

• Logic Gates: Implementation of `if/elif/else` structures for precise data sorting.

• Error Prevention: Normalizing file paths and handling case-sensitivity (e.g., `.JPG` vs `.jpg`).

How to Use:

1. Set the `path` variable in the script to the folder you want to clean.

2. Run the script in PyCharm or the terminal.

3. Watch the console log as files are moved in real-time.
